Basics: root systems, Weyl groups, lattices 

 

Base(type,n) 

returns the base of a root system 

-type: simple Lie type 

-n: posint 

> Base(B,3);
 

1&comma;−1&comma;0&comma;0&comma;1&comma;−1&comma;0&comma;0&comma;1 (1.1)
 

 

coroot(r) 

returns 2r/<r,r> as a list 

-r: list, not all entries 0 

> map(b->coroot(b),Base(B,3));
 

1&comma;−1&comma;0&comma;0&comma;1&comma;−1&comma;0&comma;0&comma;2 (1.2)
 

 

FWeight(type,n) 

returns the fundamental weights of a root system as a list of lists 

-type: simple Lie type 

-n: posint 

> FWeight(B,3);
 

1&comma;0&comma;0&comma;1&comma;1&comma;0&comma;12&comma;12&comma;12 (1.3)
 

 

RWeylGroupGen(type,n) 

returns the generators of the Weyl group of a root system as an orthogonal matrix group as a list of matrices. 

-type: simple Lie type 

-n: posint 

> RWeylGroupGen(B,3);
 

(1.4)
 

 

ZWeylGroupGen(type,n) 

returns the generators of the Weyl group of a root system as an integer matrix group as a list of matrices. 

-type: simple Lie type 

-n: posint 

> ZWeylGroupGen(B,3);
 

(1.5)
 

 

HighestRoot(type,n) 

returns the highest root of a root system 

-type: simple Lie type 

-n: posint 

> HighestRoot(B,3);
 

1&comma;1&comma;0 (1.6)
 

 

ROrbit(type,n,omega) 

returns the orbit of omega in R^n under the action of the Weyl group (orthogonal) as a list of lists 

-type: simple Lie type 

-n: posint 

-omega: list 

> ROrbit(B,3,[-1,0,0]);
 

−1&comma;0&comma;0&comma;0&comma;−1&comma;0&comma;0&comma;0&comma;−1&comma;0&comma;0&comma;1&comma;0&comma;1&comma;0&comma;1&comma;0&comma;0 (1.7)
 

 

RPull(type,n,omega) 

returns the unique dominant weight containing omega in R^n in its orbit under the action of the Weyl group as a list 

-type: simple Lie type
-n: posint 

-alpha: list 

> RPull(B,3,[-1,0,0]);
 

1&comma;0&comma;0 (1.8)
 

 

ZOrbit(type,alpha) 

returns the orbit of alpha in Z^n under the action of the Weyl group (integer) as a list of lists 

-type: simple Lie type 

-alpha: list 

> ZOrbit(B,[-1,0,0]);
 

−1&comma;0&comma;0&comma;1&comma;−1&comma;0&comma;0&comma;1&comma;−2&comma;0&comma;−1&comma;2&comma;−1&comma;1&comma;0&comma;1&comma;0&comma;0 (1.9)
 

 

Pull(type,alpha) 

returns the unique vector in N^n containing alpha in Z^n in its orbit under the action of the Weyl group (integral representation) as a list 

-type: simple Lie type 

-alpha: list 

> Pull(B,[-1,0,0]);
 

1&comma;0&comma;0 (1.10)
 

 

VertexFundomCoefficients(type,n) 

returns the coefficients of the vertices of the fundamental alcove in the basis of fundamental weights as a list. 

The last entry of the list (n+1) is always 1 for convenience. Think of it as the coefficient of 0. 

-type: simple Lie type 

-n: posint 

> VertexFundomCoefficient(B,3);
 

1&comma;2&comma;1&comma;1 (1.11)
 

 

VertexFundom(type,n) 

returns the vertices of the fundamental alcove as a list of lists. 

The last entry of the list (n+1) is 0. 

-type: simple Lie type 

-n: posint 

 

> VertexFundom(B,3)
 

1&comma;0&comma;0&comma;12&comma;12&comma;0&comma;12&comma;12&comma;12&comma;0&comma;0&comma;0 (1.12)
