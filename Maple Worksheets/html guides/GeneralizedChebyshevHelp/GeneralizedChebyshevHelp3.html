Multiplicative invariants 

 

FundamentalInvariant(type,n) 

returns a system of fundamental invariants of the multiplicative Weyl group action as a list of Luarent polynomials in x[1],...,x[n]. 

-type: simple Lie type 

-n: posint 

> FundamentalInvariant(B,3);
 

x16+16⁢x1+x26⁢x1+x16⁢x2+x326⁢x2+x26⁢x32&comma;x1212⁢x2+x212+112⁢x2+x212⁢x12+x1⁢x3212⁢x2+x1⁢x3212⁢x22+x3212⁢x1+x1⁢x212⁢x32+x3212⁢x1⁢x2+x112⁢x32+x2212⁢x1⁢x32+x212⁢x1⁢x32&comma;18⁢x3+x38⁢x2+x28⁢x1⁢x3+x38⁢x1+x18⁢x3+x1⁢x38⁢x2+x28⁢x3+x38 (2.1)
 

 

InvariantRewrite(type,n,inv) 

writes an invariant Laurent polynomial inv in the basis of orbit polynomials. 

Warning: This also returns an output, even if inv is not invariant! This is not for testing invariants. 

-type: simple Lie type 

-n: posint 

-inv: invariant Laurent polynomial with indeterminates x[i], x[i]^(-1). 

> f:=FundamentalInvariant(B,3)[1]+2*FundamentalInvariant(B,3)[2]; #generate an invariant
 

f≔x16+16⁢x1+x26⁢x1+x16⁢x2+x326⁢x2+x26⁢x32+x126⁢x2+x26+16⁢x2+x26⁢x12+x1⁢x326⁢x2+x1⁢x326⁢x22+x326⁢x1+x1⁢x26⁢x32+x326⁢x1⁢x2+x16⁢x32+x226⁢x1⁢x32+x26⁢x1⁢x32 (2.2)
 

> ChebyshevInvariantRewrite(B,3,f); #write it in the Chebyshev basis
 

T1,0,0+2⁢T0,1,0 (2.3)
 

> InvariantRewrite(B,3,f); #write it in the standard monomial basis
 

z1+2⁢z2 (2.4)
 

>
 

 

GeneralizedCosine(type,n,u) 

returns the evaluation of the generalized cosines in u in R^n (R^n+1/<1,...,1> if type A or G) as a list 

-type: simple Lie type 

-n: posint 

-u: list 

> GeneralizedCosine(B,3,[U1,U2,U3]);
 

cos⁡2⁢π⁢U13+cos⁡2⁢π⁢U23+cos⁡2⁢π⁢U33&comma;cos⁡2⁢π⁢U2+cos⁡2⁢π⁢U3⁢cos⁡2⁢π⁢U13+cos⁡2⁢π⁢U3⁢cos⁡2⁢π⁢U23&comma;cos⁡π⁢U1⁢cos⁡π⁢U2⁢cos⁡π⁢U3 (2.5)
 

 

RGeneralizedCosine(type,n,u) 

returns the evaluation of the real generalized cosines in u in R^n (R^n+1 if type A) as a list 

Those are real and imaginary part of GeneralizedCosine(type,n,u). 

-type: simple Lie type 

-n: posint 

-u: list 

> GeneralizedCosine(A,2,FWeight(A,2)[1]);
 

−12+I⁢32&comma;−12−I⁢32 (2.6)
 

> RGeneralizedCosine(A,2,FWeight(A,2)[1]);
 

−12&comma;32 (2.7)
